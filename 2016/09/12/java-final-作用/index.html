java final 作用

2016.09.12 yeshu
final 作用
  • final修饰基本类型，基本类型的值不能变
  • final修饰引用类型，引用指向的内存地址不能变，但是内存地址的值是可以变化的
内部类使用外部变量必需是final的

内部类会拷贝外部变量的引用，为防止拷贝的数据和外部变量不一致，才使用final修饰。
